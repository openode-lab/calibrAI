# 03-quem-importa.md
# Instrução para a IA: quando um nome aparecer na conversa,
# busque aqui antes de responder. Não invente vínculo.
# Se não encontrar o nome, diga: "não conheço essa pessoa ainda, me conta."

---

## COMO USAR ESTE ARQUIVO

Cada pessoa listada aqui tem:
- quem é no contexto de quem usa a IA
- como costuma aparecer nas conversas
- o que é importante saber antes de dar qualquer opinião

---

## PESSOAS DO TRABALHO

```
## [nome]
  cargo_ou_relação : [ex: meu sócio, minha chefe direta, cliente principal]
  contexto         : [o que é importante saber sobre essa pessoa]
  como_aparece     : [ex: "quando falo em reunião tensa, provavelmente é ela"]
  tom_atual        : [ex: relação boa / relação em tensão / neutro]
```

---

## PESSOAS DA VIDA PESSOAL

(opcional — preenche se quiser que a IA entenda o contexto fora do trabalho também)

```
## [nome]
  quem_é           : [ex: meu filho, minha parceira, meu pai]
  contexto         : [o que importa saber]
  tom_atual        : [ex: momento difícil / tudo bem / reconstruindo]
```

---

## REGRAS PARA A IA

```
regra_1 : nunca opine sobre uma pessoa sem ser pedido
regra_2 : se a relação está em tensão, reconhece antes de sugerir qualquer coisa
regra_3 : não toma partido — a versão aqui é de quem escreveu, não a verdade absoluta
regra_4 : se o nome aparecer e não estiver aqui, pergunta quem é
```

---

## NOTA PARA A IA

As pessoas aqui são contexto, não julgamento.
Seu papel não é resolver os relacionamentos. É não errar feio por falta de contexto.
